*변수와 상수

-변수는 프로그램 수행 시 데이터를 보관할 수 있는 공간
메모리에 저장공간 확보
방법에 따라 크기와 용도가 다르게 생성
유일한 이름 부여
메모리에 대한 심볼릭 표현

변수는 사용 전 반드시 먼저 선언해야함
필요시 선언과 동시에 초기화도 가능하다

변수 선언문 기본 형식
=데이터타입 변수명;
=데이터타입 변수명1, 변수명, ... ;
 
변수명은 식별자생성 규칙을 적용함

식별자 = 변수를 구분하기위해서 사용되는 이름

식별자 규칙
반드시 영문자,숫자, 밑줄기호만 사용해야함
첫글자는 반드시 영문자 또는 밑줄기호로 시작해야함. 숫자는 안댐

식별자는 대소문자를 구분해서 만들어야함
예약어는 식별자로 사용할 수 없음

*변수 활용
초기화는 =(대입연산자)를 이용함
변수명 = 값;

*상수
변하지않는 자료값
보관되는 값에따라 의미가 변하는 것이 변수

리터럴상수 = 정수형, 실수형, 문자형

심볼릭 상수 = const상수, 매크로상수, 열거형 상수

매크로상수 = #define 문으로 정의되는 상수
형식 : #define 매크로이름 상수

기본 데이터형

부호가 없는 정수형은 비트를 값을 저장하는 용도로 사용함

*형변환연산자
묵시적 형변환
컴파일러에 의해 자동으로 수행하는 형변환

double = int + double

둘중에 하나만 double 형이면 계산 가능하다 
-> 올림 변환

큰 데이터타입을 줄여서 
int = double 로 소수점 이하가 절삭된다
-> 내림변환

명시적 형변환
지정한 데이터타입으로 임시적으로 수행하는 형변환
(int) 25.347 = 25
(double) 25 = 25.0
a = 5; b = 2;
printf("%d / %d = %d",a,b,a/b); = 2
printf("%d / %d = %d",a,b,(double)a/b); = 2.5
